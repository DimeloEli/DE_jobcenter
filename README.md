# DE_jobcenter

Simple and easy to install third eye Job Center script
- ESX 1.9+ compatible
- OX Compatible
- Configurable
- Open source

# Dependencies
- [es_extended](https://github.com/esx-framework/esx-legacy)
- [ox_lib](https://github.com/overextended/ox_lib)
- [ox_target](https://github.com/overextended/ox_target)
